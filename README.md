# LootGen - Генератор игрового лута 🎲

## 📝 Описание проекта
Консольная утилита для симуляции выпадения предметов из сундуков в видеоиграх.  
Проект создан в рамках курса "Инструментальные средства разработки ПО".

## 🛠 Технологии
- Python 3.11+
- Flake8 (линтинг)
- Black (форматирование)
- GitHub Actions (CI/CD)
